# Meu Projeto Projeto de certificação 1 -notas-atletas

Este é um projeto é uma aplicação capaz de receber o nome e as notas de um atleta, calcular a média e apresentar ao usuário

## Funcionalidades
*   O aplicativo busca recuperar e exibir no console as notas recebidas pelos Atletas, Seu nome, suas notas e a média válida:
*   Foi criada a função obterNotas: Através de um for percorri a matriz, dai ordenei estas notas utilizando .sort "notasOrdenadas", exibindo de forma crescente as notas, e facilitando a exclusão das notas maior e menor utilizando o .slice e armazenando numa variavel "NotasMeio", depois disso utilizei .reduce para somar estas notas e armazenei numa variável "somaNotas", após criei uma variável para receber a o quociente de somaNotas pelo tamanho da matriz notasMeio.length "media". exibindo tudo no cosole.log().
